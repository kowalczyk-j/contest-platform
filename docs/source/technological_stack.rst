Stack technologiczny
---------------------

* **Backend:** Python Django
    * Cache: Redis
    * Kolejki: Dramatiq
    * Harmonogramowanie: yacron
* **Frontend:** React + Vite
* **Baza danych:** PostgreSQL
* **Testy automatyczne:** Selenium
* **Testy jednostkowe:** Unittest
* **Konteneryzacja:** Docker
* **Skrypty:** Shell
* **Dokumentacja:** Sphinx
